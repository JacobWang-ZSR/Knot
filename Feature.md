# 项目背景

## 项目名称: Knot

## 项目目的：设计一个在nas运行的docker程序，可以通过web网页访问得到密码信息。并且也能用过web页面记录新的密码信息。

# 实施细节

## 数据读取：密码存放在nas的硬盘的一个.jason文件中，可以通过dock的yaml文件设置密码的路径信息。

## 数据格式：jason的数据格式，包含的关键字如下：
    1. keyName: 用户名
    2. keyValue: 密码
    3. targetName: 什么场景的密码，为字符串
    4. targetUrl: 访问场景的地址，为字符串
    5. Other_x: 其他信息，其中x是数字，这个是附加信息，可以有Other_1, Other_2等，也可以没有

## web网页要求
    1. 可以切换主题颜色，深色和浅色模式。按钮放在右上。用一个logo表示切换主题风格。
    2. 可以展示出不同的jason数据，每个数据为一个带圆角的矩形方块。矩形相同大小。上侧显示targetName(一种背景):上侧的内容使用尽量少的高度一个行高，如果长度不够显示targetname则使用尽量少的行数；下方(另一个背景)显示keyName, keyValue以及其他信息。下方如果没有更多信息，下方内容的背景色占满剩余的区域。keyValue需要点击一下才显示，否则显示为*;
    2.5. 每个不同的数据块，在上侧的targetName的最右边，增加一个按钮，点一下进入修改模式，可以修改这个数据块中所有的数据内容，包括keyName, keyValue, targetName等。
    3. 可以有搜索功能，搜索默认匹配targetName，也可以通过下拉菜单选择别的key：keyName,targetUrl。
    4. 网页上有添加新密码的功能,在搜索功能的右边，有一个加入新密码的按钮，按下后，出现新增密码的section，并且网页滚动到这个section,使得section能被完全看到。滚动1秒。
    5. 有选择jason文件的地址的功能,需要文件浏览功能。同时如果修改了的话，需要保持该地址。这个按钮放在右上角，切换主题按钮的边上。点一下才会再页面上方出现这个功能块。点完确认后，多出来的功能（选择文件什么的）都隐藏，并且更新页面的密码信息。
    6. 打开网页的时候，有密码的保护才能进入页面，默认密码123456，并且提示在网页上。并且有修改密码的功能，在进入网页的右上有一个按钮，可以修改密码。这个按钮有一个锁的icon
    7. 需要有删除某个密码的命令，在每个密码的最右侧有一个打叉按钮。点一下经过确认后，可以删除这个密码。
